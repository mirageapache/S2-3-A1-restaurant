## 🎯專案驗收目標
✅使用Express框架建立<br>
✅讀取JSON檔案，將資料載入應用程式<br>
✅使用Express-handlebars樣版引擎，並動態呈現資料<br>
✅使用Params 實作動態路由<br>
✅使用Query String 實作搜尋功能<br>


## 🔗開始使用

1. Clone這支Repository
```
git clone https://github.com/mirageapache/S2-3-A1-restaurant.git
```
2.安裝Express
```
npm install express
```
3.啟動程式
```
npm run start
```
4.瀏覽網頁
```
開啟瀏覽器並輸入http://localhost:3000，即可瀏覽網頁
```
5.終止程式
```
在終端機按下兩次 ctrl + C 即可終止程式
```
